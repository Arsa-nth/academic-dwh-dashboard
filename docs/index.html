Tentang Project

Project ini mengimplementasikan Data Warehouse untuk mengkonsolidasikan data nilai akademik mahasiswa yang sebelumnya tersebar di berbagai file spreadsheet Excel menjadi satu Single Source of Truth.

Pipeline ETL dibangun dengan Python, mengekstraksi data dari 4 file Excel (3 presensi praktikum + 1 nilai teori), mentransformasikan dengan pembersihan data (null handling, #DIV/0! removal), dan memuatnya ke dalam Star Schema di database relasional.

  • 57 mahasiswa, 740 fact records
  • 3 mata kuliah: Metode Statistika I & II, Metode Sains Data I
  • 10 komponen penilaian (6 praktikum + 4 teori)
  • Privasi dijaga — semua nama & NIM dianonimkan di output publik
